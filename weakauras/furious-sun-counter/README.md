## Furious Sun Counter

Shows the # of times `Furious Sun` has procced since the start of combat (T18
class trinket `Sacred Draenic Incense`, WW proc).

**Classes**: All

**Requested by**: Cherry

### Changes

#### v0.1 (2016-03-26):

Fixed an issue where the counter could spuriously increment if an RSK cast
caused combat.


#### v0.0 (2016-03-25):

Initial release.
