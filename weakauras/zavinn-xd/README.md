## Zavinn xd

Whispers "xd" to Zavinn on BNet every 30 minutes. Change `xdInterval` in the
`Actions > Init` custom code to alter the interval.

**Classes**: All

**Requested by**: Arc

### Changes

#### v0.1 (2016-04-09):

Fixed an issue that caused the WeakAura to stop working when the computer was
restarted.

#### v0.0 (2016-03-27):

Initial release.
